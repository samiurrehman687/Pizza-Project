Pizza Ordering Website 

A fully functional pizza ordering website with secure user authentication and admin management. Currently deployed on PythonAnywhere.

Features

User Features
- Sign Up & Login: Secure authentication system
- Password Reset: Users can reset password via email
- Account Activation: Activation email sent to new users
- Customer Dashboard: View order history, manage profile
- Place Orders: Order pizzas and customize toppings
- Under Construction Notice: Pages under maintenance display proper message

Admin Features
- Admin Panel: Manage users, orders, and menu items
- Logging: Track user activity and errors
- Order Management: View, update, or delete orders

Security & Middleware
- Secure Middleware: Protect sensitive pages (customer/admin dashboards) 
- Login Required Middleware: Users must be logged in to access specific pages
- Input Validation & Error Handling: Ensures safe and smooth experience
- Mixins : for restrict admin to access customer dashboard.

Technologies Used
- Python 3.13  
- Django latest  
- HTML, CSS, js
- MySQL
